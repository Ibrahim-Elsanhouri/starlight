Organization Chart View

A new hierarchical visualization view for Odoo

This module introduces a new Organization Chart (org) view type, allowing any hierarchical model to be displayed as an interactive tree structure. It is ideal for models that follow a parent/child pattern such as departments, categories, employees, tasks, menus, or any custom nested structure.

Features

  • New view type: org
  • Clean and interactive hierarchical visualization
  • Automatically detects root and child nodes
  • Supports any Odoo model with parent/child relationships
  • Customizable node templates and styling
  • Highlights selected nodes and related links
  • Seamless integration into standard Odoo views
